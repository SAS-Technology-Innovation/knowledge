## Signing In to Windows

### Windows Login

<Steps>
  <Step title="Power On Computer">
    Press power button and wait for Windows to load
  </Step>

  <Step title="Enter Credentials">
    **Username:** Your SAS username (may include domain: SAS\username)

    **Password:** Your SAS password
  </Step>

  <Step title="Press Enter or Click Sign In">
    Windows will authenticate and log you in

    Profile loads with your settings and applications

```text Domain Login (if required)

Password: your-sas-password
```

## Connecting Windows to SAS WiFi

### WiFi Setup on Windows

<Steps>
  <Step title="Click WiFi Icon">
    Click the WiFi icon in the system tray (bottom right corner)
  </Step>

  <Step title="Select SAS Network">
    Choose **SAS** or **SAS-Secure** from the list
  </Step>

  <Step title="Enter Credentials">
    **Username:** Your SAS username (without @sas.edu.sg)

    **Password:** Your SAS password
  </Step>

  <Step title="Accept Certificate">
    Accept the security certificate when prompted
  </Step>

  <Step title="Connect">
    Click OK/Connect and wait for connection
  </Step>
</Steps>

<Accordion title="Advanced WiFi Settings (if needed)">
  **Manual Configuration:**

  - **Security type:** WPA2-Enterprise
  - **Encryption:** AES
  - **Authentication:** PEAP or TTLS
  - **Identity:** SAS username (no @sas.edu.sg)
  - **Password:** Your SAS password

  **Certificate:**
  - Trust the SAS certificate when prompted
</Accordion>

## Troubleshooting Windows Issues

<AccordionGroup>
  <Accordion title="Can't sign in to Windows" icon="circle-exclamation">
    **Login problems:**

    1. **Verify username format:**
       - Try: `username`
       - Or try: `SAS\username`
       - Use SAS password

    2. **Check for caps lock** - Password is case-sensitive

    3. **Try different keyboard layout** - Ensure English keyboard selected

    4. **Password expired?** - Contact IT to reset

    5. **Account locked?** - Too many failed attempts - contact IT

    **Contact IT:** [ithelp@sas.edu.sg](mailto:ithelp@sas.edu.sg)
  </Accordion>

  <Accordion title="Slow performance" icon="gauge-simple-low">
    **Windows running slow:**

    **Quick Fixes:**
    1. **Restart computer** - Clears memory and temporary issues
    2. **Close unused programs** - Check Task Manager
    3. **Check for updates** - Install pending Windows updates
    4. **Scan for malware** - Run Sophos scan

    **If still slow:**
    - Clear browser cache
    - Check disk space (should have 20GB+ free)
    - Contact IT for deeper investigation
  </Accordion>

  <Accordion title="Sophos/Security alerts" icon="shield-exclamation">
    **Security software issues:**

    See the [Sophos Security Guide](/guides/security/sophos) for detailed information.

    **Quick actions:**
    - Don't ignore security alerts
    - Let Sophos quarantine threats
    - Contact IT if repeatedly seeing alerts
    - Don't try to disable security software
  </Accordion>

  <Accordion title="Can't access network resources" icon="network-wired">
    **Network drive or printer issues:**

    1. **Verify WiFi connection** - Connected to SAS network?
    2. **Restart computer** - Reconnects network resources
    3. **Check credentials** - May need to re-enter SAS password
    4. **Map drive manually** - IT can provide server paths

    **Persistent issues:** Contact [ithelp@sas.edu.sg](mailto:ithelp@sas.edu.sg)
  </Accordion>
</AccordionGroup>
